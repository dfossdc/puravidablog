# LinkedIn Featured — Paste-Ready Copy for Each Item

Each link takes ~90 seconds of clicking through LinkedIn's modal. I've already filled in the URLs/titles/descriptions for you — just copy-paste from this doc.

## Setup

1. Go to <https://www.linkedin.com/in/puravidachiropractic/details/featured/>
2. Click the **+** at the top right of the Featured section.
3. Pick **Add a link**.
4. For each item below: paste URL → click "Add" → paste Title → paste Description → **upload the matching social card image (path provided)** → click **Save**.
5. Repeat 6 times.

LinkedIn does not auto-fetch title/description from your URLs (we tested), and the auto-pulled OG images are too small. Each item below has a **matching branded 1200×630 social card** I generated — upload it via the pencil icon under "Thumbnail" in the Add Link dialog.
